SQL Logical Operators

SQL logical operators are used to test for the truth of the condition. A logical operator, like the comparison operator, returns a boolean value of TRUE, FALSE, or UNKNOWN. Logical operators are used to combine or manipulate the conditions given in a query to retrieve or manipulate data. Here are some of the logical operators in SQL, such as OR and AND.

Some of the Operators with Syntax

AND

Selects records where both conditions are true.

SELECT * FROM EMPLOYEE WHERE SALARY < 10000 AND AGE > 25;

OR

Selects records where at least one of the conditions is true.

SELECT * FROM STUDENTS WHERE RANK > 20 OR MARKS > 600;

NOT

Selects records where the condition is not true.

SELECT * FROM EMP WHERE NOT SALARY = 10000;

Ordering Operation

ASC

Sorts the result set in ascending order.

SELECT * FROM EMP ORDER BY ID ASC;

DESC

Sorts the result set in descending order.

SELECT * FROM EMP ORDER BY ID DESC;

Grouping Operation

Groups rows that have the same values in specified columns into aggregate data.

SELECT FNAME, LNAME FROM EMP GROUP BY NAME;

Aggregate Functions (Mathematical Operations)

AVERAGE

SELECT AVG(SALARY) FROM EMP;

MINIMUM

SELECT MIN(SALARY) FROM EMP;

MAXIMUM

SELECT MAX(SALARY) FROM EMP;

SUM

SELECT SUM(SALARY) FROM EMP;

COUNT

SELECT COUNT(ID) FROM EMP;

DISTINCT

SELECT DISTINCT(NAME) FROM EMP;

ROUND

SELECT ROUND(MARKS, 2) FROM STUDENTS;

String Functions (String Handling Functions)

LENGTH

SELECT LEN('APPLE');

REVERSE

SELECT REVERSE('APPLE');

REPLICATE

SELECT REPLICATE('APPLE', 3);

UPPERCASE

SELECT UPPER('apple');

LOWERCASE

SELECT LOWER('APPLE');

CONCATENATION

SELECT CONCAT('NE', 'PA', 'L');

LEFT MOST CHARACTER

SELECT LEFT('APPLE', 3);

RIGHT MOST CHARACTER

SELECT RIGHT('APPLE', 3);

Other Operators

IN

Selects values within a given set of values.

SELECT ADDRESS FROM STD WHERE ADDRESS IN ('KTM', 'BIKE', 'PTN');

BETWEEN

Selects values within a given range.

SELECT SALARY FROM EMP WHERE SALARY BETWEEN 3000 AND 40000;

PERCENTAGE

Used with LIKE to search for a specified pattern in a column.

SELECT NAME FROM STD WHERE NAME LIKE '%a';

UNDERSCORE

Used with LIKE to search for a specified pattern in a column.

SELECT NAME FROM STD WHERE NAME LIKE '%__B';

HAVING

Used to filter records that meet the criteria after an aggregation.

SELECT NAME FROM STD WHERE GRADE = 'A' HAVING MARKS > 800;

These are some of the examples of logical operators for a quick review.
